AsyncMessage GameCarrier.Adapter.GcClientConnection.ConnectAsync ( int  timeout = 10000)

Asynchronous version of Connect. Asynchronous command will complete in one of following case:

  • OnConnectEvent - on successfull connect
  • OnConnectionErrorEvent - on connect failure

  • timeout - none of events above occured during timeout

    var operation = await client.ConnectAsync(timeout: 2000).ExecuteAsync();

    if (operation.rt.Exception != null) // exception occured during connect throw operation.rt.Exception;

    if (operation.rt.WasTimedOut) // connect timeout throw new Exception("connect timeout"); if (operation.rt.WasCompleted) // connect succeeded { }

◆ OnMessageEvent

EventHandler<MessageEventArgs> GameCarrier.Adapter.GcClientConnection.OnMessageEvent

Event, occured on receive message from Server. Event argument provides multiple ways to work with message data:

  • args.Prt & args.Length - as unmanaged pointer
  • args.RawData - as managed byte array
  • args.Msg.Buffer - a controlled memory buffer (message data) that can be used for reading and writing (SafeBuffer)
  • args.Msg.AsStream - access to unmanaged blocks of memory (message data) from managed code as Stream (UnmanagedMemoryStream)
  • args.Msg.AsAccessor - random access to unmanaged blocks of memory (message data) from managed code (UnmanagedMemoryAccessor)
