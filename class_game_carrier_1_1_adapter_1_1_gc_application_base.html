Detailed Description

Base class for Game Carrier application. Application configured in Game Carrier's config.json file in "apps" section. config.json file also contains virtual hosts configuration ("vhosts" section), each virtual host configured to:

  • listen specific port on specific protocol
  • accept incoming connections and redirect them to GcApplicationBase.OnConnect() callback
    • OnConnect method should create GcConnectionBase instance
    • OnConnect method may analyze ConnectionInfo argument to distinguish connects by port
  • receive connection messages and redirect them to GcConnectionBase.OnMessage() callback

Inherit your application class from GcApplicationBase, provide implementation for OnStart, OnConnect, OnStop, OnShutdown methods.
