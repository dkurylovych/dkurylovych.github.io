Detailed Description

Managed Counter class providing functionality of Game Carrier in-memory counters

  • Create counter in your GcApplicationBase.OnStart() callback, provide counter name
  • call Increment() / Decrement() / Add(n) methods to modify counter value
  • obtain current counter value using Value property
