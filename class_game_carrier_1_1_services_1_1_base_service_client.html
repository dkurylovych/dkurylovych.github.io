Detailed Description

Base class for Service Client. Create Service Client class, and:

  • inherit from BaseServiceClient
  • implement Service Interface
  • pass ServiceName in .ctor
  • register event handlers in .ctor by calling RegisterEventHandler / RegisterEventHandler<T>
  • implement Client Interface methods in Async style
    • Request-Response style methods implemented by chaining calls CreateRequest(...) + ExpectResponse<T>()
    • one-way Event style method implemented by calling SendEvent()
