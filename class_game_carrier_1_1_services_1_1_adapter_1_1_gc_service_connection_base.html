Detailed Description

Base class for service connection (client session). Create ServiceConnection class, and:

  • inherit from GcServiceConnectionBase
  • return new instance of connection in GcServiceBase.OnConnect() method
  • provide ServiceName in .ctor
  • register operation handlers for service by calling RegisterOperationHandler / RegisterOperationHandler<T>
  • operation handler receive OperationContext parameter, and, optionally - argument
    • call context.SendResponse from handler - to send Response to Client
    • call SendEvent - to send Event to Client immediatelly
    • call context.SendEvent - to send Event to Client after Response was sent
