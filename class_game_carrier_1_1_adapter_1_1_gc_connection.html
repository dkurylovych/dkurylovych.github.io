AsyncMessage GameCarrier.Adapter.GcConnection.DisconnectAsync ( int  timeout = 10000)

Asynchronous version of Disconnect. Asynchronous command will complete in one of following case:

  • OnDisconnectEvent - on successfull disconnect

  • timeout - none of events above occured during timeout

◆ OnDisconnectEvent

EventHandler<DisconnectEventArgs> GameCarrier.Adapter.GcConnection.OnDisconnectEvent

Event, occured on disconnect. Event argument provides disconnect reason information. OnDisconnectEvent on Client will be invoked in 2 cases:

  • server initiate disconnect - client will receive OnDisconnectEvent
  • client call Disconnect - client also will receive OnDisconnectEvent

◆ OnGroupMessageEvent

EventHandler<GroupMessageEventArgs> GameCarrier.Adapter.GcConnection.OnGroupMessageEvent

Event, occured on receive group-message from Client. Event argument provides multiple ways to work with message data:

  • args.Prt & args.Length - as unmanaged pointer
  • args.RawData - as managed byte array
  • args.Msg.Buffer - a controlled memory buffer (message data) that can be used for reading and writing (SafeBuffer)
  • args.Msg.AsStream - access to unmanaged blocks of memory (message data) from managed code as Stream (UnmanagedMemoryStream)
  • args.Msg.AsAccessor - random access to unmanaged blocks of memory (message data) from managed code (UnmanagedMemoryAccessor)
